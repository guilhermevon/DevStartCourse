# 💻 Programming for Your First Internship

> A beginner-friendly online learning platform designed to teach the fundamentals of Full Stack web development and help aspiring developers land their first internship.

This project consists of a complete educational website featuring an interactive course, student area, pricing page, instructor profile, and contact page, all built with a responsive and modern design.

---

## 📂 Project Structure

```bash
📁 index.html       → Landing page
📁 course.html      → Student area with course lessons
📁 about.html       → Instructor information
📁 pricing.html     → Pricing plans
📁 contact.html     → Contact page
📁 style.css        → Website styling
📁 script.js        → JavaScript functionality
📁 README.md        → Project documentation
```

---

## 💻 Technologies

* **HTML5**
* **CSS3** (Flexbox, Grid & Media Queries)
